# BossCV

## 介绍

BossCV是一个待在Boss消息框旁边的AI对话助手插件，协助你快速回复HR的消息！

核心特性：

- 隔离式会话管理：每一个不同的HR消息队列对应一个单独的AI对话框
- 自动获取岗位信息：点击[发送岗位信息]按钮，会自动抓取网页中的岗位信息并发送给AI

准备实现的特性：

- 智能调用（Agent）：为AI配备工具箱，允许其在回答问题时，先判断是否需要额外信息，再自动选择性读取您编辑的简历和Boss聊天框中的信息进行回答。

## 如何使用？

BossCV是一款浏览器插件，目前只支持Edge浏览器，尚未上架到插件市场。使用它，你需要先点击扩展，进入到扩展页面后，点击【加载解压缩的扩展】，然后选择项目根目录下的`edge_extension`文件夹。

![image-20260625222709796](https://raw.githubusercontent.com/Just-A-Freshman/image-bed/main/Typora/image-20260625222709796.png)

随后，你只需要简单地配置一下大语言模型的API。点击扩展中的【Boss直聘AI面试助手】配置对应的AI模型，并配置三项信息即可。

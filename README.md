# BossCV

## 介绍

BossCV是一个住在Boss消息框旁边的AI对话助手插件，协助你快速回复HR的消息！

![image-20260624214300442](https://raw.githubusercontent.com/Just-A-Freshman/image-bed/main/Typora/image-20260624214300442.png)

## 如何使用？

BossCV是一款浏览器插件，目前只支持Edge浏览器，尚未上架到插件市场。使用它，你需要先点击扩展，进入到扩展页面后，点击【加载加压缩的扩展】，然后选择项目根目录下的`edge_extension`文件夹：

![image-20260624214558777](https://raw.githubusercontent.com/Just-A-Freshman/image-bed/main/Typora/image-20260624214558777.png)

随后，你只需要简单地配置一下大语言模型的API。点击扩展中的【Boss直聘AI面试助手】配置对应的AI模型：

![image-20260624214811014](image/image-20260624214811014.png)
